# 🎵 Club Run - Clean Repository

## 🚀 Overview

Club Run is a revolutionary music education platform that combines AI-powered learning with community-driven development. This repository contains the clean, GitHub-compatible version of our codebase.

## 🛠️ Technical Stack

### Frontend
- **Framework**: React + TypeScript
- **Build Tool**: Vite
- **Styling**: CSS Modules
- **State Management**: React Context

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Database**: Prisma ORM
- **Authentication**: Google OAuth
- **Real-time**: WebSocket

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Git

### Installation

1. **Clone the repository**
   ```bash
   git clone [your-github-repo-url]
   cd club-run-clean
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

4. **Start development server**
   ```bash
   npm run dev
   ```
